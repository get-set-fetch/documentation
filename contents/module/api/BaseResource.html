BaseResource(siteId, url, createDefaultPlugins)

Persistence class representing a resource within a site. When creating a new site, its url is automatically added as a resorce. At site level, a resource is selected for crawling, crawled, processed and updated. If the current resource links to other resources, those are persisted for future crawling, thus enabling a recursive crawl of the entire site.
